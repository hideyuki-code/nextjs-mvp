# 🌐 フェーズ3：SEOと構造化データの学習（10h）

## 📋 フェーズ概要

SEOの基本概念を理解し、Next.jsでのメタデータ管理と構造化データの実装を学習します。
**ハンズオン形式**で実際にコードを書きながらSEO最適化を進めます。

## 🎯 学習目標

- [ ] CSR・SSR・ISR・SSGの違いとSEOへの影響を理解
- [ ] Next.js metadata APIの使い方を理解
- [ ] 構造化データ（JSON-LD）の実装
- [ ] SEO測定ツールの使い方を理解
